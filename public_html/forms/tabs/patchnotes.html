Patch 2.0.0 (Current)

Release Date: 15/05/2018

First live patch.

Features:

Auto You can copy your order from eFood's final order grid before order submission and paste it using the order input button. The program will automatically generate the items of the order into the Order Items box. You can drag and drop each item on the user who bought it and by pressing the Calculate button a result window will pop up with the discounted prices per user. In addition you can always reload your calculated order by pressing the Restore Last Order button, or clear all the order items from the users by pressing the Reload button.

Old School Inspired from everyone's favourite Peinata Helper desktop app, with this mode you can calculate your order if for any reason, using the auto mode is not possible. Firstly you have to input the Total Price which will automatically calculate your order prices. Then apply the price for each user individually, you can also use addition 3.5+4.2+0,55 for example and the price will be calculated automatically. Note that the Calculate button will remain disabled if the total price is less than 15â‚¬ or less than 2 users have prices. By clicking the Calculate button a result window will pop up with the discounted prices per user.

Settings Here you can setup your users, each table of the app will generate using all the users saved in this form.

Extras Every result window has the option of tipping. The tip added will be split equally to each user that took part in this order and the new final price will be displayed.
